# Entra ID

Scripts, tools, and fixes related to **Microsoft Entra ID** (formerly Azure Active Directory).

## Subfolders

| Folder | Description |
|--------|-------------|
| [Users](./Users/) | User account creation, modification, and queries |
| [Groups](./Groups/) | Security groups, Microsoft 365 groups, dynamic groups |
| [Conditional-Access](./Conditional-Access/) | Conditional Access policies and named locations |
| [App-Registrations](./App-Registrations/) | App registrations, service principals, and permissions |
| [RBAC](./RBAC/) | Role-Based Access Control and custom role definitions |

| Folder | Description |
|--------|-------------|
| [Onboarding](./Onboarding/) | New user creation, license assignment, and welcome automation |
| [Offboarding](./Offboarding/) | Account deactivation, data export, and license reclaim |
| [Bulk-Operations](./Bulk-Operations/) | CSV-driven bulk create, update, or delete operations |
| [Licensing](./Licensing/) | License assignment, reporting, and optimization |

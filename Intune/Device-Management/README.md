# Intune – Device Management

Scripts and tools for querying, managing, and reporting on **enrolled Intune devices**.

## Available tools

- [`Get-IntuneDeviceInfoTool.ps1`](./Get-IntuneDeviceInfoTool.ps1) – Data functions for querying Intune managed devices from Microsoft Graph. Dot-sourced by the M365 Tool Portal (`BASE/MAIN/Start-M365ToolPortal.ps1`).

## Examples of what belongs here

- PowerShell scripts to list all enrolled devices
- Scripts to retire, wipe, or sync devices remotely
- Reports on device compliance status
- Queries for devices with specific OS versions
